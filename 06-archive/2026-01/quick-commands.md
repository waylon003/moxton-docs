# 三端并行快捷命令

> **更简单的调用方式** - 无需每次写完整的 Task 代码

---

## 方式 1: 直接描述（推荐）

直接说需求，我会自动识别并并行启动子代理：

```
实现商品评价功能，需要后端API、前端页面、后台管理
```

```
修复购物车价格问题，需要三端协作
```

```
优化订单查询性能，后端添加索引，前端和后台同步优化
```

---

## 方式 2: 简化关键词

使用关键词模式：

```
并行开发: 商品评价
```

```
三端协作: 修复订单状态更新
```

```
同步更新: 用户字段变更
```

---

## 方式 3: Skill 调用

```
/parallel 商品评价功能开发
```

```
/triple 修复购物车价格
```

```
/all-projects 更新用户信息字段
```

---

## 方式 4: 结构化描述

```
### 新功能：商品评价

**后端** (moxton-lotapi):
- Review 表: id, productId, userId, rating, comment
- API: GET/POST/PUT/DELETE /reviews

**前端** (nuxt-moxton):
- 评价列表组件
- 评价表单组件
- 星级评分组件

**后台** (moxton-lotadmin):
- 评价管理页面
- 审核功能
```

实现商品收藏功能，包括后端API和前端页面
