## 🎯 使用指南

### 开始集成

1. **阅读 API 文档** - 在 [02-api/](../02-api/) 查看相关接口
2. **查看集成指南** - 参考对应模块的集成文档
3. **检查依赖关系** - 在 [04-projects/DEPENDENCIES.md](../04-projects/DEPENDENCIES.md) 确认依赖

### 添加新指南

创建新的集成指南时，请包含：

1. **概述** - 集成目标和范围
2. **前置条件** - 需要的准备
3. **步骤说明** - 详细的集成步骤
4. **代码示例** - 实际代码演示
5. **测试验证** - 如何验证集成成功
6. **常见问题** - FAQ
